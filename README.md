# My Python Journey - Πανελλαδικές & Projects

Εδώ ανεβάζω τον κώδικα που γράφω όσο προετοιμάζομαι για τις Πανελλαδικές (ΕΠΑΛ) και τα πρώτα μου δικά μου projects.

### Τι θα βρείτε εδώ:
- **School Exercises:** Λύσεις από το σχολείο.
- **Algorithms:** Βασικοί αλγόριθμοι που μαθαίνω στην ύλη (Sorting κλπ).
- **Projects:** Μικρά tools που φτιάχνω για να λύνω καθημερινά προβλήματα.
    - *Grades Calculator:* Υπολογισμός Μ.Ο. για 4 μαθήματα ενός μαθητή με έλεγχο εγκυρότητας βαθμών και αξιοποίηση f-strings.
    - *TODO Task Manager:* Υλοποίηση json για την αποθήκευση αριθμημένων tasks σε ένα αρχείο και την επεξεργασία τους, επιτρέπωντας στον χρήστη να τα επισημάνει ως True ή False.

### Το Setup μου:
- Python 3.10
- VS Code
- Git για version control

*Στόχος είναι να περάσω στη σχολή που θέλω και να χτίσω σωστές βάσεις στον προγραμματισμό.*
